# To-Do List App

A Java console application to manage 
your daily tasks.

## Features
- Add new tasks
- View all tasks with numbering
- Delete tasks by number
- Simple menu-driven interface

## How to Run
1. Install Java JDK
2. Open in IntelliJ IDEA
3. Run ToDoList.java

## Technologies Used
- Java
- ArrayList (Collections)
- Scanner (user input)
- Switch-case menu
- Do-while loop
